Reference

POST /register - Creates a new user by providing a username, email, and password.

POST /login - Authenticates a user using their email and password.

POST /newChallenges - Creates a new challenge with details (text, description, dataset, image, result).

GET /challenges/:challengeId - Retrieves details of a specific challenge using its ID.

DELETE /deleteChallenge/:challengeId - Deletes a challenge specified by ID.

GET /all-challenges - Retrieves all registered challenges.

GET /my-challenges?userId=:userId - Retrieves challenges of a user using their ID.

GET /challenges/:challengeId - Retrieves details of a specific challenge using its ID.
